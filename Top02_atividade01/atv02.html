Introdução ao curso

Este é o cruso de Programação para Web, nele, iremos ser intoduzidos aos conceitos básico das ferramentas utilizadas para contruir páginas online.
Para isso iremos ter aulas no formato de vídeo-aulas e atividades semanais para reforçar os conteúdos vistos nas aulas, além disso, teremos um
projeto a ser desenvolvido ao longo do semestre que irá abordar todo o conteúdo visto.
